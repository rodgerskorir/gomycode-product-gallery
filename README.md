# 🛍️ Product Gallery with Theme Toggle

A responsive 3-card product showcase built with **HTML, CSS, and JavaScript** featuring:

- 🌗 Light/Dark mode toggle
- 🎯 Hover effects on cards
- 🎬 Smooth fade-in animations
- 🚫 No frameworks used
